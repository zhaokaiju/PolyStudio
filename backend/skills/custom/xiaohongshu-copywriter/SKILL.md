# 小红书文案创作助手

## 概述

本技能专注于小红书平台内容创作，帮助生成符合小红书社区调性的高质量文案。涵盖：好物推荐、种草笔记、干货攻略、穿搭分享、美妆教程、生活方式、探店打卡等多种内容类型。

## 核心工作流程

### 1. 需求确认

收到创作请求后，确认以下信息（如用户未提供，主动询问）：
- **内容主题**：产品/服务/话题是什么？
- **目标受众**：面向哪类人群？（年龄、兴趣、痛点）
- **内容类型**：种草推荐 / 干货教程 / 生活方式 / 探店打卡 / 其他
- **核心卖点**：最想传达的1-3个关键信息
- **语言风格**：活泼可爱 / 专业知性 / 接地气 / 其他

### 2. 内容结构

标准结构：标题 → 开头钩子 → 正文 → 互动引导 → 话题标签

### 3. 标题规范

- 12-18个字（含emoji算1字），关键词前置
- 禁用极限词：最、第一、绝对、100%等

### 4. 开头钩子

必须满足之一：痛点切入 / 反常识 / 具体场景
禁止：「大家好」「今天给大家分享」

### 5. 话题标签

3-5个，结构：大类词 + 垂直细分词 + 场景词
