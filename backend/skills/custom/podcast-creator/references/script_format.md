# 脚本格式示例

## 多人对话播客格式

```
【多人对话播客】
主持人：大家好，欢迎来到今天的节目，我是主持人小林...
嘉宾：很高兴能来到这里，大家好，我是XXX...
主持人：今天我们请到了XXX，来聊聊关于AI的话题...
嘉宾：对，这个话题我非常感兴趣，因为...
主持人：那您觉得AI对普通人的生活影响最大的地方是什么？
嘉宾：我觉得是效率层面，比如...
...
主持人：好的，感谢XXX今天的分享，我们下期再见！
嘉宾：谢谢大家，再见！
```

## 独白播客格式

```
【独白播客】
第1段：大家好，我是XXX，今天想和大家分享一个关于...的话题。
第2段：首先，我们来谈谈背景。在过去的几年里，...
第3段：那么，为什么这件事如此重要？原因在于...
第4段：接下来，我想分享三个实用的建议...
第5段：第一点是...
第6段：第二点是...
第7段：第三点是...
第8段：最后，希望今天的分享对你有所帮助，我们下次见。
```

## 有声书格式

```
【有声书 - 第一章】
旁白：话说在很久很久以前，有一个小镇...
角色A（小明）：妈妈，我今天能出去玩吗？
角色B（妈妈）：可以，但要在天黑前回来。
旁白：小明高兴地跑出了门，向着森林的方向奔去...
角色A（小明）：哇，这片森林好美啊！
旁白：就在这时，他发现了一个奇怪的东西...
```

## 广播剧格式

```
【广播剧】
旁白：夜幕降临，城市陷入了一片寂静...
侦探：（皱眉）这个案子不简单。
助手：您是说，凶手不止一个人？
侦探：正是。看这里——（停顿）——这个脚印的间距，说明...
助手：（惊呼）天哪，那岂不是说...
旁白：两人对视，空气中弥漫着紧张的气氛。
```

## 脚本编写建议

- **角色标注清晰**：每段对话前明确标注角色名
- **段落不要太长**：每段话控制在 2-4 句以内，符合真实口语节奏
- **留有停顿空间**：可在脚本中用（停顿）、（思考）等标注语气节点
- **情绪提示**（可选）：可为角色加注情绪标注，如（激动地）、（低声）
- **时长估算**：普通语速约每分钟 200-250 字，可据此估算总时长
