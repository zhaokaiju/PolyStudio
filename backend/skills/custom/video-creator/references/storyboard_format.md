# 分镜脚本格式示例

## 标准格式

```
我将为您生成一个30秒的[主题]视频，分镜如下：

镜头1（5秒）：[详细场景描述，包含视觉元素、动作、情绪]
镜头2（5秒）：[详细场景描述]
镜头3（5秒）：[详细场景描述]
镜头4（5秒）：[详细场景描述]
镜头5（5秒）：[详细场景描述]
镜头6（5秒）：[详细场景描述]

整体风格：[统一风格描述，如：温暖写实、赛博朋克、水彩动画等]
色调：[整体色调描述，如：暖橙色调、冷蓝色调等]
```

## 示例：科技产品宣传片（30秒）

```
我将为您生成一个30秒的科技产品宣传视频，分镜如下：

镜头1（5秒）：黑色背景中，产品从远处缓缓浮现，光线从四周汇聚照亮产品轮廓，营造神秘感
镜头2（5秒）：产品正面特写，表面材质细节清晰可见，蓝色科技光效流动扫过
镜头3（5秒）：使用者的手拿起产品，背景是现代简约的办公环境，自然光
镜头4（5秒）：产品屏幕亮起，界面UI展示，操作手势特写，流畅动效
镜头5（5秒）：使用者面带微笑俯视产品，情绪：满足、专注，背景虚化
镜头6（5秒）：产品logo缓缓出现，背景光效扩散，品牌感收尾

整体风格：现代科技感，写实摄影风格
色调：冷蓝色调为主，金属质感，局部暖光点缀
```

## 示例：自然风景短片（15秒，空镜）

```
镜头1（5秒）：清晨山间薄雾，阳光穿透树梢，光束倾斜射入，树叶轻微摇动
镜头2（5秒）：溪流从石块间流过特写，水面反光闪烁，流动感强
镜头3（5秒）：山顶全景，云海翻涌，远处山峰若隐若现，静谧壮观

整体风格：自然纪录片风格，高清写实
色调：清晨暖金色
```

## 示例：有角色出镜的故事短片（20秒）

```
（前提：对话历史中已有角色"小雨"的参考图）

镜头1（5秒）【空镜】：城市街道，傍晚，霓虹灯开始亮起，人流穿梭
镜头2（5秒）【有角色-小雨】：小雨走在街道上，背影，步伐轻快，手提购物袋
镜头3（5秒）【有角色-小雨】：小雨转身看向橱窗，侧脸特写，眼神专注
镜头4（5秒）【空镜】：橱窗内的展示品特写，灯光映照，质感细腻

注：镜头2、3含角色，使用 edit_volcano_image_tool 基于小雨参考图生成
    镜头1、4为空镜，使用 generate_volcano_image_tool 生成
整体风格：城市写实，电影质感
色调：暖橙色霓虹调
```
